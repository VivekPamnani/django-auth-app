General instructions:
  1. You must participate in this study using a laptop or a desktop.
  2. This study requires you to have a real keyboard.
  3. Before you start participating, you will be required to take a two-part screening test on the home page of the website.
    • The first part of the screening test is to make sure you clear the eligibility criteria for the study.
    • The second part of the screening test is to check if you are color blind. If you are color blind, you cannot participate in this study.
